Utility Classes
===============

Utils
-----
Static utility methods for common game operations:

  Utils.GetCollisionForce(Collision collision)           // Returns impact force
  Utils.GetTeamColor(PlayerTeam team)                    // Color (blue or red)
  Utils.GetNameFromTeam(PlayerTeam team)                 // "Blue", "Red", etc.
  Utils.GetTeamFromName(string name)                     // Inverse of GetNameFromTeam
  Utils.GetNameFromRole(PlayerRole role)                 // "Attacker", "Goalie"
  Utils.GetRoleFromName(string name)                     // Inverse of GetNameFromRole
  Utils.GameUnitsToMetric(float value)                   // Convert to meters
  Utils.GameUnitsToImperial(float value)                 // Convert to feet/inches
  Utils.Vector2Clamp(Vector2 value, Vector2 min, Vector2 max)
  Utils.Vector3Clamp(Vector3 value, Vector3 min, Vector3 max)
  Utils.RotatePointAroundPivot(Vector3 point, Vector3 pivot, Vector3 angles)
  Utils.WrapEulerAngle(float angle)
  Utils.Map(float value, float from1, float to1, float from2, float to2)

Common usage examples:
  // Convert game units to human-readable
  float speedMph = Utils.GameUnitsToImperial(puck.Speed);

  // Get team-specific color for UI
  Color teamColor = Utils.GetTeamColor(player.Team);

  // Clamp a position to the rink bounds
  Vector3 clamped = Utils.Vector3Clamp(pos, minBound, maxBound);

  // Map a value range (useful for UI sliders)
  float opacity = Utils.Map(distance, 0f, 10f, 1f, 0f);

NetworkingUtils
---------------
Helpers for resolving network references:

  NetworkingUtils.GetPlayerFromNetworkObjectReference(NetworkObjectReference)
  NetworkingUtils.GetPlayerPositionFromNetworkObjectReference(NetworkObjectReference)
  NetworkingUtils.GetPuckFromNetworkObjectReference(NetworkObjectReference)

These convert NetworkObjectReference values (received via events or RPCs) into
their corresponding game objects. NetworkObjectReference is a serializable
reference to a NetworkObject that works across the network.

Usage:
  void OnPuckSpawned(Dictionary<string, object> msg)
  {
      if (msg.TryGetValue("puck", out var ref) && ref is NetworkObjectReference nor)
      {
          Puck puck = NetworkingUtils.GetPuckFromNetworkObjectReference(nor);
          if (puck != null) Debug.Log("Puck spawned at " + puck.transform.position);
      }
  }
