Enums
=====

Player Enums
------------
  PlayerPhase { None, TeamSelect, PositionSelect, Play, Replay, Spectate }

  PlayerPhase indicates where the player is in the game flow:
    - None: Not yet in a game or connecting
    - TeamSelect: Choosing a team (Blue/Red/Spectator)
    - PositionSelect: Choosing a role (Attacker/Goalie)
    - Play: Actively on the field playing
    - Replay: Watching a goal replay
    - Spectate: Spectating (not playing)

  PlayerTeam { None, Blue, Red, Spectator }

  PlayerRole { None, Attacker, Goalie }

  PlayerHandedness { None, Left, Right }

  Handedness affects which side the player holds their stick. Left-handed
  players hold the stick with their left hand lower.

Game Enums
----------
  GamePhase { None, Warmup, PreGame, FaceOff, Play, BlueScore, RedScore,
              Replay, Intermission, GameOver, PostGame }

  Represents the current state of the match (see GameMode System for details).

  ApplicationQuality { Low, Medium, High, Ultra }
  ShadowQuality { Low, Medium, High, Ultra }
  Units { Metric, Imperial }

Server/Connection Enums
-----------------------
  ConnectionPhase { Disconnected, Connecting, Connected }
    Stages of establishing a connection to a server.

  ConnectionRejectionCode { Unreachable, ServerFull, TimedOut, Banned, NotWhitelisted, MissingPassword, InvalidPassword, MissingMods, ServerStarting, Unknown }
    Reasons a connection request was rejected.

  DisconnectionCode { ConnectionLost, Disconnected, Kicked, Banned }
    Reasons for disconnection (timeout, kicked, etc.).

  ReconnectionPhase { None, AwaitingPassword, AwaitingMods }
    States during reconnection to a server.

  AuthenticationPhase { None, Authenticating, Authenticated }
    Stages of Steam authentication during connection.

  TransactionPhase { None, Starting, Started }
    Phases of Steam workshop item transactions.

  ServerReadinessPhase { NotReady, Starting, Ready }
    Server readiness state. Starting while mods are loading, Ready once done.

  SteamWorkshopItemPhase { None, Downloading, Updating, Installed }
    States of a Steam workshop item being downloaded/installed.

UI Enums
--------
  UIPhase { None, LockerRoom, Playing }
    Overall UI state machine phase.

  AppearanceCategory { Head, Body, Stick }
    Categories in the appearance customization screen.

  AppearanceSubcategory { Headgear, Flags, Mustaches, Beards, Jerseys, StickSkins, StickShaftTapes, StickBladeTapes }
    Subcategories within each appearance category.

  QuickChatCategory { Information, Compliments, Reactions, Apologies, Emojis }
    Categories of quick chat messages.

  HeadgearRole { Attacker = 1, Goalie = 2, Any = 4 }
    Which player role a headgear item is for (flags-based).

  JerseyTeam { Blue = 1, Red = 2, Any = 4 }
    Which team a jersey item is for (flags-based).

  StickSkinTeam { Blue = 1, Red = 2, Any = 4 }
    Which team a stick skin is for (flags-based).

  PlayerLegPadState { Idle, Butterfly, ButterflyExtended }
    Visual state of player leg pads.

  AvatarSize { ... }
    Size options for player avatars in UI (Small, Medium, Large).

  KeyBindInteraction { Press, Release, DoublePress, Hold, Continuous, Toggle }

  KeyBindInteractionType { Press, Hold }
    Simplified key bind interaction type.

  HierarchyChangeType { AddedToParent, RemovedFromParent, ChildrenReordered, AttachedToPanel, DetachedFromPanel }
    UI Toolkit hierarchy change event types.

  DeploymentProvider { None, Edgegap, PuckOrchestrator }
    Dedicated server deployment provider.

  DerivativeMeasurement { Velocity, ErrorRateOfChange }
    PID controller derivative measurement mode.

  TCPServerMessageType { PreviewRequest, PreviewResponse }
    LAN server browser message types.

Item Enums
----------
  Item categories (used as strings in Item.categories[]):
    "flag", "headgear", "mustache", "beard", "jersey", "stickSkin",
    "stickShaftTape", "stickBladeTape", "attacker", "goalie", "unlisted"

  Items can belong to multiple categories. For example, a headgear might be
  tagged with both "headgear" and "goalie" to indicate it's goalie-only.
